# QR Code Generator

## Description
A simple QR Code Generator web app where users can input text or URL and generate a QR code. The app also provides an option to download the generated QR code.

## Features
- Enter any text or URL to generate a QR code.
- Option to download the generated QR code image.
- Clean and responsive UI.

## How to Use
1. Open the app in a web browser.
2. Enter your text or URL in the input field.
3. Click on the "Generate QR Code" button to generate the QR code.
4. Click on the "Download QR Code" button to download the image.

## Technologies Used
- HTML
- CSS
- JavaScript
- QR Code API
